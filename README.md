# Sobre o Projeto

Este projeto é um exercício focado na construção de interfaces estáticas (wireframes/mockups) a partir de imagens de referência. O objetivo é reconstruir fielmente as telas da aplicação em HTML e CSS, garantindo uma estrutura com semântica adequada, layout responsivo e a aplicação rigorosa de boas práticas de acessibilidade.
